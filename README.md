# 꿀꺽이 신호 대시보드

키움 「꿀꺽이 마스터」 조건검색+차트수식 동시확정 신호를, 종목별 1분봉 차트로 보여주는 대시보드입니다.

- 표에서 **종목을 클릭**하면 그날 1분봉 차트가 뜹니다.
- 캔들: 빨강=상승 · 파랑=하락
- 거래량: 평균(이동평균)선 위로 **평소 3배 넘게 터진 봉**을 주황색 + ▼ 로 표시(비이상 거래량)
- 차트 위 **신호 · ▲매수 · ▼매도** 마커 표시
- 우측 상단 **다크모드** 토글

## 보는 법
공개 주소(GitHub Pages): `https://<본인아이디>.github.io/<저장소이름>/`

## 매일 갱신하는 법
로컬에서 봇으로 그날 데이터를 쌓은 뒤:
```
python make_dashboard.py 20260716   # 원하는 날짜
```
생성된 `reports/dashboard_YYYYMMDD.html` 을 이 저장소의 `index.html` 로 덮어써서 올리면(push/업로드) 공개 페이지가 갱신됩니다.

※ 이 페이지는 신호·차트 데이터만 담고 있으며, 매매 코드나 계좌·토큰 등 비밀정보는 포함하지 않습니다.
